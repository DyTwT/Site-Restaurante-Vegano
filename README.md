# 🌿 Green Yard - Website

Projeto de um site responsivo para restaurante vegano, desenvolvido com foco em layout moderno, organização visual e boas práticas de HTML e CSS.
!(./capa.png)

---

## 📖 Sobre o Projeto

O objetivo deste projeto foi desenvolver uma página web com:

- Estrutura semântica em HTML5
- Estilização com CSS3
- Layout utilizando Flexbox
- Design responsivo
- Organização em seções estratégicas

O site simula a página inicial de um restaurante vegano, contendo:

- 🍽 Comida em destaque
- 📚 Nosso catálogo
- ⭐ Seus favoritos
- 🎨 Identidade visual personalizada

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries (Responsividade)

---

## 📱 Responsividade

O projeto foi desenvolvido para se adaptar a:

- 💻 Desktop
- 📱 Mobile
- 📲 Tablets

---

## 🎯 Funcionalidades

- Layout em colunas
- Organização visual moderna
- Estrutura modular
- Estilização personalizada para títulos e seções
- Ajustes de alinhamento com Flexbox
